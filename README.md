# My Frontend Web App

A modern, advanced frontend website using HTML, CSS, and JS.

## Setup Instructions

1. **Build and run with Docker:**
   ```sh
   docker build -t my-frontend-app .
   docker run -d -p 80:80 --name frontend my-frontend-app
   ```

2. **Access in browser:**
   ```
   http://your-ec2-public-ip
   ```
